  - GOG
  - Epic

  - Itch.io
  - Greenman
  - Humble
  - Fanatical
- All other files under `data/` contain games free within the following gaming subscriptions
  - Xbox Game Pass
  - Playstation Plus
  - EA Play
  - Prime Gaming
  - Nintendo Online
